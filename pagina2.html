O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de documentos escritos em HTML ou XML. O CSS define como os elementos HTML devem ser exibidos, permitindo que os desenvolvedores separem o conteúdo da apresentação visual, resultando em páginas web mais organizadas e flexíveis.

Conceitos Básicos

Seletores

Seletores são utilizados para selecionar os elementos HTML que você deseja estilizar. Existem vários tipos de seletores:

Seletores de tag

Estiliza todos os elementos de um determinado tipo.

p { color: blue; }

Seletores de classe

Estiliza elementos que possuem uma determinada classe.

.minha-classe { font-size: 20px; }

Seletores de ID

Estiliza um elemento específico com um determinado ID.

#meu-id { background-color: yellow; }

Propriedades e Valores

Propriedades são características que podem ser aplicadas aos elementos HTML e os valores definem como essas propriedades serão exibidas. Por exemplo:

color: red; /* propriedade color com o valor red */

Classes e IDs

Classes e IDs são atributos que podem ser adicionados aos elementos HTML para aplicar estilos específicos:

Classe

Utilize o atributo class para aplicar um estilo a múltiplos elementos.

<div class="minha-classe"></div>

ID

Utilize o atributo id para aplicar um estilo a um único elemento.

<div id="meu-id"></div>

Cascata e Especificidade

O CSS é aplicado aos elementos HTML seguindo regras de cascata e especificidade. A cascata determina a ordem em que os estilos são aplicados e a especificidade define a prioridade dos seletores:

  • Seletores de ID têm maior prioridade que seletores de classe.
  • Seletores de classe têm maior prioridade que seletores de tag.
